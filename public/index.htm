Hello!

This is a small project I've created mainly for educational purposes. In the core of it is a database containing a table of strings and their md5 sums. The user can either search if an md5 sum is listed in the table, and if so, receive its source string, or just input a string and get its md5 sum. When that happens, the string and its md5 sum are added to the database, thus expanding it with likely strings. That by itself is a simple (and existing) mechanism, but it allowed me to create a small site that incopretates a few different languages and technologies;
  • HTML and CSS are obviously used to format and style this page.
  • Native Javascript is used in Paul Johnston's md5 implementation to calculate the sum on the client side.
  • jQuery powered AJAX is used to send the plain strings and the md5 sums to the server.
  • PHP and MySQL are used on the server to receive plain strings, calculate their sums, and store them in a database, and to check if requested md5 sums are already calculated in the database, and if so, to return them.
  • Python was used to generate the initial database.
Enjoy!
